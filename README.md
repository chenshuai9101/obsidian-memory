# Obsidian Memory 🧠

> 通用 Obsidian Vault 记忆持久化 Skill — 让任何 AI Agent 拥有跨会话长期记忆

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-green)](https://github.com/chenshuai9101/obsidian-memory)
[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)

## 概述

将 AI Agent 的工作记忆自动同步到 Obsidian Vault，实现跨会话的长期记忆持久化。

## 核心功能

- **📝 日记同步** — 每日工作记录自动写入 Obsidian 日记
- **🧠 长期记忆** — 重要决策和知识自动归档
- **📋 任务追踪** — 任务状态自动同步更新
- **🔍 全文检索** — 利用 Obsidian 搜索能力检索历史记忆

## 使用方式

```bash
# 在 OpenClaw 中
/skill obsidian-memory
```
